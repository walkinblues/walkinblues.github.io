关于分贝的概念和音响系统中的应用

发表于 2020-03-01 更新于 2022-04-30 分类于 读书笔记
《SoundSystemEngineering》读书笔记

关于分贝的概念和音响系统中的应用

分贝

  • 分贝被定义为数值为$10^{0.1}$的功率比值的对数形式。

  • 任何两个功率如果相差$N,dB$，则他们的比值为$10^{N(0.1)}$:

$$
\frac{P1}{P2} = 10^{N(0.1)}
$$

$$
10lg\frac{P1}{P2} = N\quad dB
$$

奈培

  • 奈培$(Np)$被定义为：
    $$
    ln(\frac{E1}{E2}) = N(Np)
    $$

    • E为电压值

  • 奈培$dB$之间的关系：

$$
（\frac{P1}{P2}) = (\frac{E1}{E2})^2 = (e^2)）
$$

  • 可以得到$1Np$等同于$8.686dB$

分贝的基本应用

  • 分贝始终且只能以功率比值为基础。任何其他形式的比值（比如：电压电流或声压）都必须首先平方转换成功率的比值之后再转换成分贝表示的功率级。

    • 功率比=电压比的平方：
    • （2-5）

  • dB本身是一种对数标尺刻度，术语“级”始终用在以dB为单位表示功率。（声压级、声功率级等）

  • 给出基准参考值时，才可以使用分贝数得到绝对值。

  1. 电压比到功率比

    • 在电路的同一点上电压放大倍数 = 功率放大倍数^2
    • $10log\frac{P1}{P2} = 20lg\frac{V1}{V2}$
    • 备注：Lg（2/1） = 0.30103

  2. dBV

    • 在使用分贝这一单位视，“级”是专门针对功率而言的。对于电压幅度上的增加，应该称为“放大量”。
    • dBV的基准参考电压为1.0V（1.0V时为0dBV）。

  3. dBm

    • 音频领域的功率参考基准为$10^{-3}W$
    • $0dBm = 0.001W$
    • 在定压系统中，功率级随阻抗变化。在定流系统中，电压随阻抗变化。所以当使用电压表示的分贝时阻抗必须要给出。
    • 电功率的测量:
      • $W = EIcos\theta = I^2Zcos\theta = \frac{E^2}{Z}cos\theta$
        • Z为阻抗，$\theta$为E和I的相差
        • 使用单一频率的正弦波电压和电流才成立
    • $功率变化量(dB) = 10log[(\frac{E_1^2}{E_2^2})(\frac{R2}{R1})] = 20lg\frac{E1}{E2} + 10lg\frac{R2}{R1}$
      • 仅当$R1 = R2$时，$功率级(dB) = 20lg\frac{E1}{E2}$
      • 实践当中与测量精度有关，当器件在工作时，由于温度和环境变化R值也会产生变化
    • 实例：
    • 当以dBm为单位表示功率级时，通常将$E2 = 0.775V，R2 = 600\Omega$（声频标准）。E2和R2可以是任意值，只要结合起来得到1mW的功率。
    • 功率定义：
      • $视在功率 = E\times I或E^2/Z$
      • $平均的实际功率或吸收功率为(E^2/Z)cos\theta$
      • $无功功率为(E^2/Z)sin\theta$
      • $功率因子 = cos\theta$
      • 术语“增益”或“损耗”始终表示被测设备在系统输出上的功率增益或功率损耗

  4. $L_I、L_W和L_P$

    • 声强级$L_I = 10lg\frac{xW/m^2}{10^{-12}W/m^2}$
    • 声功率级$L_W = 10lg\frac{总的声功率瓦数}{10^{-12}W}$
    • 声压级$L_P = 20lg\frac{xPa}{20\mu Pa} = 20lg\frac{P_{rms}}{20\mu Pa}$
      • Prms是声压的均方根值，单位是Pa
    • 平方反比定律

      • 分析半径为0.282m的球体（该球表面积为$1m^2$），球中心有一个全指向的点声源，向外辐射1W的声功率。

      • 此时在0.282m处$L_P = L_I = L_W = 120dB$

      • 球的表面积与半径平方成正比，将球半径加倍，则表面积为原来的4倍（$S = 4\pi r^2$）

      • 半径增加一倍，在0.564m处，声强降低6dB，0.282处的四分之一。$L_P,L_I,L_W$都降低6dB。

      • 这种变化称为平方反比定律、

      • 方向性因子（$Q$)

        • 当声源被视为半球形辐射时，在0.282处$L_P = 123.01dB，L_W$维持120dB。这3.01dB的变化体现的是单位面积上功率的2:1变化。
          • 半球形辐射子的方向性因子时全方向辐射子的2倍。
